# Qazo-Media-Widget

Удобный и визуально приятный виджет для управления медиа сессиями в Windows, поддерживающий работу с несколькими мониторами и расширенное управление через системный трей.

## Особенности
- **Мульти-мониторная поддержка**: Корректно работает и перемещается на любой активный монитор.
- **Индикатор и управление громкостью**: Удобное управление системной громкостью с помощью колесика мыши.
- **Функциональность системного трея**:
  - Левый клик: Play/Stop для активной медиа сессии.
  - Правый клик: Контекстное меню с опциями:
    - Свернуть/Развернуть приложение.
    - Переместить виджет на другой дисплей (с выбором монитора).
    - Включение/отключение "Автозапуска".
    - Закрыть приложение.
- **Приятный интерфейс**: Анимированные фоновые элементы (пузыри) и плавная бегущая строка для длинных названий треков.
- **Технологии**: Написано на C# с использованием WinUI 3 (Windows App SDK).

## Установка и запуск
1. Перейдите на страницу **Releases** и скачайте `MediaWidgetWinUI.exe`.
2. Запустите загруженный файл. Виджет автоматически появится в нижнем левом углу основного монитора или в системном трее. Установка не требуется.
3. В системном трее выберите нужный монитор для отображения или включите автозапуск при старте Windows.
